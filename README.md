# MIXXXO
> Jigsaw 랜섬웨어 복호화 툴 개발 프로젝트 (2019.04 ~ 2019.06)
- Project by team 'Flosvia 2019'

## 목표
  1. Jigsaw 랜섬웨어의 공격(암호화) 방식을 알아본다.
  2. Jigsaw 랜섬웨어로 인해 암호화된 파일을 복호화하는 응용프로그램을 제작한다.

## 사용 기술
  1. UI
    * Python Tkinter
    * WinDLL

  2. 내부 함수
    * C
      - OpenSSL
      - WinAPI
